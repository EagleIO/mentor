MSU I.O Student Mentorship Program User Manual

5 Program Roll-Out

This page primarily serves as a procedural overview, so future Eagle I.O cohorts can implement and manage the Mentorship program.

Procedurally, Eagle I.O members:

  1. Guage interest and availability via administering a survey sent by the program director in July
    • Several follow-up attempts are made during the summer
    • Template documents:3
      • To Mentors
      • To First Year Mentees
      • First Two Events
      • Alumni Recruitment
  2. Eagle I.O consultants match interested mentors and mentees
    • This is done prior to orientation
  3. Matches are introduced before orientation by email and discuss future events (see Chapter 4)
  4. The program is monitored periodically (see Chapter 3.4 “Eagle I.O Responsibilities”)
